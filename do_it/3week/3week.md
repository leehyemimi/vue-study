## 3-2  뷰 컴포넌트

### 컴포넌트란? 

조합하여 화면을 구성할 수 잇는 블록(화면의 특정영역)을 의미



#### 전역 컴포넌트
 
여러인스턴스에서 공통으로 사용할수있다

```javascript
Vue.component('컴포넌트 이름',{
	//컴포넌트 내용
});
```

#### 지역 컴포넌트

특정인스턴스에서만 유요한 범위를 갖고있다

```javascript
new Vue({
    components :{
        '컴포넌트 이름' : 컴포넌트 내용
    }
})
```
